# Açaí Landpage

Uma landing page moderna e responsiva para uma empresa de entrega de açaí, desenvolvida com HTML, CSS e JavaScript puro.

## Recursos

- Design responsivo que funciona em todos os dispositivos
- Botão de WhatsApp para contato direto
- Código limpo e bem organizado

## Como usar

1. Clone ou baixe este repositório
2. Abra o arquivo `index.html` em qualquer navegador moderno
3. Personalize as informações conforme necessário

## Personalização

- **WhatsApp**: Atualize o número de telefone no link do botão no arquivo `index.html`
- **Cores**: Personalize as cores no arquivo `styles.css` (variáveis CSS no início do arquivo)

## Requisitos

- Navegador moderno (Chrome, Firefox, Safari, Edge)
- Conexão com a internet (para carregar fontes e o Google Maps)
